# Lead Management System

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

## Features
- Add leads
- View leads
- Edit leads
- Delete leads
- Validation included

## How to Run

### Backend
cd backend
npm install
node Backend/server.js

### Frontend
go to index.html click on run and
Open index.html in browser

## Assumptions
- Data stored in memory
- No authentication added
